- Hash maps for frequency/lookup problems
- Sliding window for subarray/substring problems

### Optimization Tips
- Always consider time and space complexity
- Use `unordered_map` for O(1) average lookup
- Prefer shifting over swapping in sorting (fewer operations)
- Use prefix sums for range queries
